# 终端开源工具链扩展清单（替代 GUI）

本文档用于补充“可脚本化替代 GUI”工具，默认按收益优先集成。

## 已集成入口

- 安装脚本：`scripts/install-cli-tools.ps1`
- 任务入口：`Justfile`
  - `just tools-install`
  - `just tools-install-minimal`
  - `just hooks-install-precommit`
  - `just hooks-install-lefthook`
  - `just ci-local`
- 提交钩子配置：
  - `.pre-commit-config.yaml`
  - `lefthook.yml`

## 工具与用途

- `fzf`：交互式筛选，适合长列表选择（对象/请求号/文件）
- `gum`：终端交互组件（输入、选择、确认），可增强脚本 UX
- `jq` / `yq`：处理 JSON/YAML 配置与结果输出
- `delta`：增强版 `git diff` 渲染
- `bat`：增强版 `cat`（语法高亮）
- `act`：本地跑 GitHub Actions，减少远端调试成本
- `duckdb`：本地结构化分析日志和结果文件
- `sqlite-utils`：快速写入/查询 sqlite，适合执行记录落盘

## 推荐落地顺序

1. 先装最小集：`fzf/gum/jq/yq`
2. 再接提交流程：`pre-commit` 或 `lefthook`（二选一为主）
3. 最后补本地 CI 与分析：`act/duckdb/sqlite-utils`

## 注意事项

- `act` 依赖 Docker；未安装时会失败。
- `pre-commit` 与 `lefthook` 建议主用一个，避免重复执行检查。
- 企业机器若禁用 `winget`，请改为手工安装并保持命令名可用。
